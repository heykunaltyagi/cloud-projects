# Contributing to Cloud Projects

Thank you for helping improve this monorepo! Contributions are welcome from every experience level.

## How to get started

1. Fork the repository.
2. Create a branch with a descriptive name:
   ```bash
      git checkout -b feature/azure-key-rotation-docs
   ```
3. Make a small, focused change.
4. Commit with a clear message.
5. Open a pull request against `main`.

## What we welcome

- Bug fixes and issue resolutions
- Documentation improvements
- Terraform module enhancements
- New automation or CI improvements
- Testing and validation updates
- Help with `README.md`, `ROADMAP.md`, and project docs

## Issue process

Before creating a new issue:

- Search existing issues to avoid duplicates.
- If you find a matching issue, add a comment or reaction.
- Use one of the issue templates in `.github/ISSUE_TEMPLATE/`.

### Good first issues

Look for issues labeled `good-first-issue`.
These are smaller, self-contained tasks that are great for new contributors.

## Pull request process

- Open a PR from your branch into `main`.
- Include a short description of the change and why it matters.
- Reference any related issue using `Fixes #123` or `Closes #123`.
- Keep PRs focused and small when possible.
- Run `terraform fmt -recursive` in any Terraform directories you change.
- If you update docs, make sure they are clear and easy to follow.

## Review and feedback

- Reviews are collaborative, not personal.
- Expect reviewers to ask questions or request small adjustments.
- If you disagree with feedback, discuss it politely in the PR.

## Non-code contributions

You can also help by:

- Improving documentation or examples
- Writing issue reports with reproducible steps
- Suggesting roadmap items
- Reviewing issue templates and workflows

## Support for new contributors

If you feel intimidated, start small:

- Fix a typo in documentation
- Add a short example to `README.md`
- Suggest a clarifying improvement to the project structure

This repo is intended to be friendly and approachable. Every contribution helps.
